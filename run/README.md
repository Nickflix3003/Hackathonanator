Azterra runner for the local PR loop.

Usage:
- Default paths: `python run/azterra.py --dry-run`
- Custom repo: `python run/azterra.py --repo C:\path\to\azterra --dry-run`
- Custom todo: `python run/azterra.py --todo C:\path\to\todo.yaml --dry-run`
- Custom loop script: `python run/azterra.py --run-loop C:\path\to\run_loop.py --dry-run`
- Re-run failed tasks: `python run/azterra.py --rerun-failed`
- Reset existing branches: `python run/azterra.py --reset-branch`
